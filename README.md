# Handy Bunny

<br/>

Ruby Version ruby-2.7.4

<br/>

Installation

    git clone https://github.com/arwiemendoza/final_project.git

    cd final_project

For Windows:

    sudo service postgresql start

For Mac:

    brew services start postgresql

    bundle install

    rails db:migrate

    rails db:seed

For Windows:

    Open database.yml
    Uncomment username and password

For Mac:

    Open database.yml
    Comment username and password

    rails s
